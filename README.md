🔤 First Non-Repeating Character (Python)
📌 Problem Statement

Given a string, find the first non-repeating character and return its index.
If all characters repeat, return -1.

🚀 Features
Takes user input
Finds first unique character
Displays:
Index of character
Character itself
Handles edge cases (no unique character)
🧠 Approach
Use a dictionary (hash map) to store frequency of each character
Traverse the string again
Return the index of the first character with frequency = 1
